# Indian Retail Analysis

## Overview

This project performs exploratory data analysis on retail sales data to identify business trends, customer purchasing behavior, and key performance metrics.

## Objectives

- Analyze retail sales data
- Clean and preprocess datasets
- Identify sales trends
- Visualize business insights
- Support data-driven decision making

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## Project Structure

```
indian retail analysis/
│── sale.py
└── README.md
```

## Results

The analysis provides valuable insights into customer behavior, product performance, and sales trends that can support business growth.
