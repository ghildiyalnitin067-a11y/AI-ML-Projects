# Customer Churn Prediction

## Overview

This project predicts whether a customer is likely to leave a company using classification algorithms. It demonstrates a complete machine learning pipeline from preprocessing to model evaluation.

## Objectives

- Analyze customer behavior
- Preprocess and clean data
- Train classification models
- Predict customer churn
- Evaluate model accuracy

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure

```
customer churn prediction/
│── data/
│── chum-prediction.py
└── README.md
```

## Results

The project helps identify customers who are at risk of leaving, enabling businesses to develop effective retention strategies.
