# 📘 Machine Learning Basics

This repository contains my hands-on implementation of fundamental Machine Learning algorithms and evaluation techniques using Python and Scikit-Learn.

The goal of this project is to understand the mathematical intuition, implementation, model training, hyperparameter tuning, and performance evaluation of regression and classification algorithms.

---

##  Topics Covered

### Regression Algorithms
- Linear Regression
- Ridge Regression (L2 Regularization)
- Lasso Regression (L1 Regularization)

### Classification Algorithms
- Logistic Regression

---

##  Model Evaluation Metrics

### Regression Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score (Coefficient of Determination)

### Classification Metrics
- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC-AUC Score
